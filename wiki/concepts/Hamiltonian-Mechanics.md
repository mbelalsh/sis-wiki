# Hamiltonian Mechanics

## One-Line Intuition

Pick a scalar function `H(q,p)` over phase space (the "Hamiltonian", usually total energy). The system's time evolution is given by the [[Symplectic-Gradient]] of `H` — moving along this vector field automatically conserves `H`.

## Why This Was Invented

William Hamilton (1830s) reformulated Newtonian mechanics to give a more unified, coordinate-free description applicable beyond simple particle motion. The reformulation generalizes naturally to:

- Systems with constraints (no need for ad-hoc Lagrange multipliers in many cases)
- Continuous and quantum systems (Hamilton–Jacobi → Schrödinger equation)
- Statistical mechanics (`H` becomes the energy in `e^{−βH}`)
- Symplectic geometry (the modern coordinate-free framing)

For NN-based dynamics modeling, Hamiltonian mechanics matters because it gives **a clean structural way to enforce conservation laws**: parameterize `H`, derive dynamics, conservation is automatic. This is the foundation for [[Hamiltonian-Neural-Network]] and its descendants.

## The Math

**Phase space.** A system with `N` degrees of freedom has phase space coordinates `(q,p) ∈ ℝ^{2N}`, where `q = (q₁, …, q_N)` are generalized positions and `p = (p₁, …, p_N)` are conjugate momenta.

**Hamiltonian.** A scalar function `H : ℝ^{2N} → ℝ`. For most mechanical systems, `H = T(p) + V(q)` (kinetic + potential energy).

**Hamilton's equations** (canonical equations of motion):
```
dq/dt = ∂H/∂p     dp/dt = −∂H/∂q
```

**Conservation.** `dH/dt = (∂H/∂q)(dq/dt) + (∂H/∂p)(dp/dt) = (∂H/∂q)(∂H/∂p) − (∂H/∂p)(∂H/∂q) = 0`. So `H` is a constant of motion along trajectories.

**Liouville's theorem.** The flow generated by Hamilton's equations preserves phase-space volume. Equivalently: any Hamiltonian time-evolution map `(q₀,p₀) → (q_t,p_t)` is bijective (volume-preserving → invertible). This makes Hamiltonian dynamics intrinsically reversible — exploited in HNN's free-reversibility property.

**Poisson bracket.** For two functions `f, g` on phase space:
```
{f, g} = (∂f/∂q)(∂g/∂p) − (∂f/∂p)(∂g/∂q)
```
Hamilton's equations can be written as `df/dt = {f, H}` for any observable `f`. The bracket structure encodes the symplectic geometry of phase space.

### Code Correspondence

```python
import numpy as np
from scipy.integrate import solve_ivp

def hamiltonian_flow(dHdq, dHdp, x0, t_span):
    """Integrate ẋ = (∂H/∂p, -∂H/∂q) from x0 over t_span."""
    def rhs(t, x):
        q, p = np.split(x, 2)
        return np.concatenate([dHdp(q, p), -dHdq(q, p)])
    return solve_ivp(rhs, t_span, x0, rtol=1e-9)

# Mass-spring: H = ½q² + ½p²  (k = m = 1)
sol = hamiltonian_flow(
    dHdq = lambda q, p: q,
    dHdp = lambda q, p: p,
    x0   = np.array([1.0, 0.0]),
    t_span = (0, 10),
)
# sol.y[0] traces a perfect circle of radius 1 in (q,p) space
```

## Toy Example

**Mass-spring (1 DoF).** `H = ½kq² + p²/(2m)`. Hamilton's equations: `dq/dt = p/m`, `dp/dt = −kq`. With `k = m = 1`: `q̈ = −q`, the simple harmonic oscillator. Phase-space trajectories are concentric circles (level sets of `H`).

**Pendulum (1 DoF).** `H = 2mgl(1 − cos q) + l²p²/(2m)`. Phase-space trajectories are nested ovals near the origin (small-angle harmonic regime); they distort and eventually break into rotating trajectories above the separatrix energy.

**Two-body (8 DoF).** `H = |p_CM|²/(m₁+m₂) + (|p₁|² + |p₂|²)/(2μ) + g·m₁m₂/|q₁−q₂|`. Conservation of `H` (energy), `p_CM` (linear momentum), and angular momentum reduces this to effectively 1 DoF (the Kepler problem).

## Connection to SiS / CTPC

Hamiltonian mechanics is the **structural foundation** for SiS architectures targeting conservative dynamics:

- **Two-body Keplerian orbits are exactly Hamiltonian.** Energy + angular momentum conservation are baked into the formulation. Equinoctial orbital elements are a canonical transformation of `(q,p)` chosen for non-singularity at zero eccentricity / inclination.
- **The hard-constraint primitive** (Hamilton's equations as architecture) becomes [[Hamiltonian-Neural-Network]] when `H` is a learned MLP.
- **Decomposition for CTPC.** Conservative core (Keplerian + zonal perturbations) → Hamiltonian. Dissipative residual (drag, SRP) → Port-Hamiltonian. The SiS philosophy splits a system into Hamiltonian-by-construction parts and learned dissipative parts.
- **Liouville reversibility** = free orbit-determination structure. Backward propagation is mathematically exact, no engineering needed.

## Open Questions

- **Canonical coordinates from observation.** When observable data isn't already in `(q,p)` form (e.g., pixel data, raw orbital position-velocity rather than position-momentum), you need a coordinate transformation that preserves the symplectic structure. Whether this transformation can be *learned* or must be hard-coded is open. HNN's pixel pendulum needed an auxiliary loss to enforce canonical structure (Eq. 7 of the paper).
- **Constrained Hamiltonian systems.** Holonomic and non-holonomic constraints add Lagrange multipliers and break the simple `(q,p)` setup. Dirac brackets generalize but are nontrivial. Relevant for spacecraft attitude (constraint: unit quaternion).
